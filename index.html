tinybpf

A minimal Python library for eBPF.

Install

uv add tinybpf --extra-index-url https://gregclermont.github.io/tinybpf/
pip install tinybpf --extra-index-url https://gregclermont.github.io/tinybpf/

With uv inline script

# /// script
# dependencies = ["tinybpf"]
# [tool.uv]
# extra-index-url = "https://gregclermont.github.io/tinybpf/"
# ///
